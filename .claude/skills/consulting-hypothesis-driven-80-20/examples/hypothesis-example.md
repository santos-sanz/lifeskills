# Example: reduce churn

## Governing question and metrics
- Question: How can we reduce churn in SMB?
- Success metrics: monthly churn rate, NRR
- Time horizon: next two quarters

## Hypotheses
1. If onboarding completion rises, then churn falls because users reach value faster.
2. If pricing tiers are simplified, then churn falls because buyers match plan to needs.
3. If support response time improves, then churn falls because issues resolve faster.

## Prioritization (80/20)
- Top hypothesis: onboarding completion
- Secondary: pricing tiers

## Tests and data needs
- Onboarding funnel drop-off by cohort
- Price plan usage by churned accounts
