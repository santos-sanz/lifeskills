# 80/20 prioritization

## Scoring dimensions
- Impact: size of upside if true
- Confidence: evidence strength or likelihood
- Effort: time and resources required
- Speed: time to learn
- Data availability: access to required data

## Simple scoring
- Score each dimension 1-5.
- Priority score = (Impact x Confidence) / (Effort x Speed).
- Use Data availability as a tie breaker.

## Sequencing
- Start with the top 1-2 hypotheses that are high impact and fast to test.
- Defer expensive tests until quick wins are exhausted.
