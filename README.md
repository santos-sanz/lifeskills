# lifeskills

Non-coding skills for Claude Code. This repo focuses on negotiation, persuasion, and consulting analysis frameworks.

Skills live in `.claude/skills/` and are written in English.

## Skills

- `negotiation-voss-tactical-empathy` - Tactical empathy negotiation based on Chris Voss.
- `persuasion-cialdini-influence-design` - Ethical persuasion using Cialdini principles.
- `pyramid-principle-structured-communication` - Structured communication using the Pyramid Principle.
- `consulting-issue-tree-mece` - MECE issue trees and problem structuring.
- `consulting-hypothesis-driven-80-20` - Hypothesis-driven analysis with 80/20 prioritization.
- `consulting-market-competition-analysis` - Market sizing and competitive landscape analysis.
- `consulting-portfolio-growth-strategy` - Portfolio review and growth strategy frameworks.

## How to use

- Invoke directly with `/skill-name <context>`.
- Or describe the task and let Claude load the skill automatically.

Each skill includes:
- `SKILL.md` with when and how to use it.
- `references/` for framework details and checklists.
- `templates/` for structured outputs.
- `examples/` for sample outputs.
