# Pyramid rules

## Core rules
- Answer first: lead with the key message.
- Use BLUF (Bottom Line Up Front) for executive scanning.
- Support with 3-5 grouped points.
- Keep groups MECE: no overlap, no gaps.
- Use parallel wording across points.
- Make each point a complete thought.

## Logic checks
- Deductive: point A and B imply C.
- Inductive: examples or facts build to a general point.
- Avoid mixing inductive and deductive in the same level.

## Clarity checks
- Each level summarizes the level below.
- Each point stands alone and is specific.
- Remove redundant or weak points.
- Replace vague nouns with concrete drivers or actions.
